# SenSante

Assistant de pré‑diagnostic médical pour le Sénégal.

## Description

SenSante utilise le *Machine Learning* pour aider au pré‑diagnostic des maladies courantes  
(paludisme, grippe, typhoïde) à partir des symptômes renseignés par le patient.

## Structure du projet

- `data/` : Données patients (CSV)
- `models/` : Modèle de Machine Learning sérialisé
- `api/` : API FastAPI
- `frontend/`: Interface web
- `notebooks/` : Scripts d’exploration
